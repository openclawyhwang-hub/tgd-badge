# tgd-badge

> Badge generator for the tGD ecosystem.

## Quick Start

```bash
pip install tgd-badge
tgd-badge "tGD" "v1.0" --color-green
```

## Usage

```
tgd-badge <label> <message> [options]
```

## Examples

```bash
tgd-badge "tGD" "v1.0" --color-brightgreen
tgd-badge "status" "active" --color-blue
tgd-badge "tGD" "compatible" --color-green --style flat
```
